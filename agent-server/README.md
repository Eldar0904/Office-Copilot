# Copilot Agent Server

## First-time setup (run once)
Open a terminal in this folder and run:
```
npm install
```

## Start the agent
```
npm start
```
You should see: `🤖 Copilot Agent running on http://localhost:3001`

## Then open the web app
Open `employee-copilot-web/index.html` in your browser.
Click the **Copilot** button → switch to **🎙️ Voice Agent** tab.
